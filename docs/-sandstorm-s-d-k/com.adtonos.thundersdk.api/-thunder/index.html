This method can be used when the process of granting consents is more complicated to save the current values. After the start method is called, the saveConsents method updates the data collection. It can be used for example to revoke the consents on the settings screen. Calling saveConsents alone is not sufficient to start the library.

The library requires certain permissions and user consents for data collection. Therefore, one of the initial screens should show the user the terms and conditions containing the necessary information about the use and processing of personal data. This is done when the user first interacts with the application, so this is a good time to call the start method with specific AdTonosConsent. Additionally the start method calls savesConsents which asks the user for the system permissions necessary for the application to work. This process is automated. The start method must be called every time the application starts. Additionally, the loadLatestConsents method can be used, which returns the most recently granted consents.

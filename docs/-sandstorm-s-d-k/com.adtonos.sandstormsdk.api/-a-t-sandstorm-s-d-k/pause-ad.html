Content copied to clipboard

Method that should be called whenever the application loses foreground. MUST BE CALLED ON UI THREAD. Its task is to perform a pause on the currently playing material. If the method succeeds true value is returned and OnVastAdPaused method of SandstormCallback is called. If the pause fails while advertising material is available, the callback OnVastError (PauseAdError) from SandstormCallback is called. In the case where the pause is due to a change in the application foreground and does not affect the playback of the advertisement, only the value false will be returned.

Return

true if pause succeeded, false if failed (probably wrong state)

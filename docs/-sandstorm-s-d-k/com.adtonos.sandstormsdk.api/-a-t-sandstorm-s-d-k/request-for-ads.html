Content copied to clipboard

The method call performs a query for the ads and then downloads them. MUST BE CALLED ON UI THREAD. When the process is complete and ready, the onVastAdsLoaded method is called. From this moment on it is possible to call playAd. The method, depending on the current state, returns the appropriate value of the SandstormAdRequestResult enum informing what is currently happening. After calling the clear method will be destroyed and their instances should not be used again. In order to be able to play the next advertisement, the playback of the previous ones must be completed. See the example in the Quick Guide to see how the whole process can be combined.
